# B.4* Gate Corpora

`elspeth_mini/` is a vendored 80-file subset copied from the local
`/home/john/elspeth/src` checkout during B.4* implementation. It includes
Protocol-heavy modules, callable dispatch maps, decorated functions, deeply
nested package paths, and third-party typed callables.

`synthetic/` is a project-local corpus with the same call-resolution patterns
and no dependency on access to the elspeth repository.
